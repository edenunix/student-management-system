# Student Management System

## Description
Web application developed in PHP and MySQL for managing students.
The system includes authentication, CRUD operations, and an admin dashboard.

## Features
- User authentication (login/logout)
- Student management (add, edit, delete)
- Admin dashboard
- MySQL database integration

## Technologies
- PHP
- MySQL
- HTML / CSS
- Bootstrap (optional)

## Project Structure
index.php → Home page
login.php → Authentication
dashboard.php → Admin dashboard
config.php → Database configuration
database.sql → Database schema


## Installation (Local)
1. Install XAMPP or WAMP
2. Import `database.sql` into phpMyAdmin
3. Place project folder in `htdocs`
4. Configure database credentials in `config.php`
5. Run `localhost/student-management-system`
